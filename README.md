# 🪪 IDCardFolio

An interactive portfolio experience inspired by the design language of modern digital ID cards.

Built to blend **personal branding, UI/UX experimentation, and web development**, this project transforms a traditional portfolio into a dynamic and immersive experience.

🔗 **Live Demo:** https://suryakiranayila.github.io/idcardfolio.github.io/

---

## ✨ Overview

IDCardFolio reimagines the standard portfolio website as an interactive digital identity card.

Designed and developed with a focus on clean aesthetics, subtle animations, and intuitive interactions, it showcases my work, skills, and creative interests through a unique visual approach.

---

## 🚀 Features

* 🪪 ID card-inspired portfolio interface
* ✨ Smooth animations and micro-interactions
* 📱 Responsive design for desktop and mobile devices
* 📄 Downloadable resume integration
* 🎨 Custom UI/UX design
* ⚡ Lightweight and fast-loading architecture
* 🌙 Modern visual styling with attention to detail

## 💻 Run Locally

Clone the repository:

```bash
git clone https://github.com/suryakiranayila/idcardfolio.github.io.git
```

Navigate to the project directory:

```bash
cd idcardfolio.github.io
```

Open `index.html` in your browser or use a local development server.

For example:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
